# -sistema-inscricao-evento- tipo land page
 Sistema de inscrição de evento com Spring Boot
Sistema de Inscrição de Evento

Este projeto é uma aplicação web desenvolvida utilizando Java com o framework Spring Boot, com o objetivo de realizar o cadastro de inscrições para eventos.

A aplicação permite que o usuário preencha um formulário com seus dados (nome, email e telefone). Após o envio, essas informações são processadas pelo backend e armazenadas em um banco de dados PostgreSQL.

A interface da aplicação foi desenvolvida utilizando HTML com o motor de templates Thymeleaf, que permite integrar páginas web diretamente com o backend do Spring Boot.

O projeto segue uma arquitetura em camadas, organizada em Controller, Service, Repository e Model, o que facilita a manutenção, organização do código e separação das responsabilidades da aplicação.

Tecnologias utilizadas no projeto:

Java
Spring Boot
Spring Data JPA
Thymeleaf
PostgreSQL
Maven
HTML e CSS

Para executar o projeto é necessário ter o Java instalado, configurar o banco de dados PostgreSQL e rodar a aplicação através de uma IDE como IntelliJ IDEA.

Após iniciar a aplicação, o sistema pode ser acessado pelo navegador no endereço:

http://localhost:8080

Este projeto foi desenvolvido com fins de estudo para praticar desenvolvimento backend com Spring Boot, integração com banco de dados e construção de aplicações web utilizando Java.
